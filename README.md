## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/dilushamadushan/todo_app_flutter.git
    ```
2. Navigate to the project directory:
    ```bash
    cd todo_app_flutter
    ```
3. Install dependencies:
    ```bash
    flutter pub get
    ```
4. Run the app:
    ```bash
    flutter run
    ```
